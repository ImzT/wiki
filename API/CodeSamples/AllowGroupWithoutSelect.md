#AllowGroupWithoutSelect

[[_TOC_]]

##About

_**Obsolete:** As of version 6.7, this setting is no longer available._

Gets or sets value indicating if it is possible to make invisible fields which are used for Visual Grouping. 
If set, fields with both "Hide This Field" (in the advanced properties) and "VG"; flags set, will not lead to report being considered as invalid.

**Default Value:** false
