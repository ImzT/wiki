#AllowInvalidCharacters

[[_TOC_]]

##About

Gets or sets the value indicating whether invalid characters in report names are allowed if not stripped. Valid characters include any character in the standard English alphabet, numbers, backslashes, underscores, single quotes, whitespace, and hyphens.

**Default Value:** false
