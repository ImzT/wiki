#AllowVirtualDataSources

[[_TOC_]]

##About

Gets or sets the availability of the [[Virtual Data Sources|/Virtualdatasource]] feature. This will do two things: it will enable the "Expose as Datasource" checkbox on the Misc tab of the Report Designer, and it will create a hyperlink on the name of any data source on the Data Sources tab which will show all visible reports that use the current data source.

**Default Value:** False
