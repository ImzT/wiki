#ConvertNullToEmptyString

[[_TOC_]]

##About

Gets or sets how null values are handled in Izenda. While False, cells which are not null but empty are treated as empty strings (zero-character string objects). Literal null cells are populated with the string '(NULL)'. When set to true, null cells are instead converted to empty strings. 

**Default Value:** False

    HttpContext.Current.Session["ReportingInitialized"] = true;
  }
}
