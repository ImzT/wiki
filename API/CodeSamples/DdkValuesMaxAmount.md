#DdkValuesMaxAmount

[[_TOC_]]

##About

This setting allows you to specify how many independent drilldownkey values are exposed in the designer. You do not have to use all of these values in each report, this just allows you to enable up to the specified number. 

**Default Value:** 2

      return;
    AdHocSettings.LicenseKey = "INSERT_LICENSE_KEY_HERE";
    //Creates a connection to Microsoft SQL Server

    Izenda.AdHoc.AdHocSettings.AdHocConfig = new CustomAdHocConfig();
    AdHocSettings.DdkValuesMaxAmount = 7; //The relevant setting
    HttpContext.Current.Session["ReportingInitialized"] = true;
  }
}
