#EqualsSelectLimit

[[_TOC_]]

##About

Gets or sets the number of records that Izenda will return from the database to populate certain filters. Filter operators which provide a list of values for the user to select from will be impacted by this limit. Examples are Equals(Select), Equals(Multiple), Equals(Checkbox), and the like. Filter operators that do not provide a list will not be impacted, such as Equals(Autocomplete), Equals(List), or Equals. 

**Default Value:** 500
