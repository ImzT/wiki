#HighChartLabelWidth

[[_TOC_]]

##About

Gets or sets the width of the label on a HighChart chart object in pixels. This is often changed when there are two or more charts being displayed on a screen and they are causing container size conflicts (meaning charts will be moved, displayed incompletely, or disappear) because of long string labels being rendered on the chart.
