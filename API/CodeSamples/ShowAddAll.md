#ShowAddAll

[[_TOC_]]

##About

Gets or sets the value indicating whether the 'Add All' buttons and dropdown should be shown on the Fields Tab of the Report Designer. Three controls will be made visible: a dropdown which allows the user to select a data source, an 'Add All' button which adds all fields from that data source, and a 'Remove All' button which removes all fields.

**Default Value:** False
