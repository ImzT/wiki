#StoreImagesToCache

[[_TOC_]]

##About

Gets or sets the value indicating how temporary images should be stored.

By default, images are saved on the disk during the rendering of reports.  For most environments, this is an acceptable method of storing images.  However, for environments that use a load balancer, or a web farm, this should be set to **false** to force Izenda to store images in the session and avoid calls to the wrong disk image.

**Default Value:** True

    HttpContext.Current.Session["ReportingInitialized"] = true;
  }
}
