#AllowLineOnBar

[[_TOC_]]

##About

**Summary:** Gets or sets whether the "Line Value" options are displayed on the Chart tab of the [[Izenda.Web.UI.AdHocReportDesigner|/API/Izenda/Web/UI/Izenda-Web-UI-AdHocReportDesigner]] when a bar chart is selected. Setting this to false will still allow the "Show Pareto" checkbox to be checked, but the line's value will not be able to be controlled. The line's value defaults to the percentage of the sum of the "Value" metric that occurred at each data point.  
**Data Type:** [[System.Boolean|http://msdn.microsoft.com/en-us/library/a28wyd50]]  
**Accepted Values:**   
**Default Value:** True
