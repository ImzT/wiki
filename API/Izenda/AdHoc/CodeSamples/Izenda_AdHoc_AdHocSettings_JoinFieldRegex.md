##About

**Summary:** Gets or sets the regular expression used for filtering fields that can be used to join tables on the Datasources Tab of the [[Izenda.Web.UI.AdHocReportDesigner|/API/Izenda/Web/UI/Izenda-Web-UI-AdHocReportDesigner]]. If the name of a column matches this expression then it will be excluded from the dropdown used to select join fields.
