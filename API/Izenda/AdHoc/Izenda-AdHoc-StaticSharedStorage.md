##Methods

###Activate(System.Boolean)
 Activates or deactivates the Static Shared Storage mechanism. This must be the very first call in InitializeReporting to fully leverage the storage capabilities. 

Parameters: 

* [[System.Boolean|http://msdn.microsoft.com/en-us/library/a28wyd50]] forceStaticMemorySetting  - Determines whether to force settings normally dependant upon the [[Izenda.AdHoc.Scope|/API/Izenda/AdHoc/Izenda-AdHoc-Scope]] setting to use static memory or not.
