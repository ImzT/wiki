##About

**Summary:**  Fires when the point is updated programmatically through the <code>.update()</code> method. The <code>this</code> keyword refers to the point object itself. One parameter, <code>event</code>, is passed to the function. The new point options can be accessed through <code>event.options</code>. Returning <code>false</code> cancels the operation.
