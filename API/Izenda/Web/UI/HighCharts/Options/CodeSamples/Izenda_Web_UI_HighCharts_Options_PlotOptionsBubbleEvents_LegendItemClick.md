#LegendItemClick

[[_TOC_]]

##About

**Summary:**  Fires when the legend item belonging to the series is clicked. The <code>this</code> keyword refers to the series object itself. One parameter, <code>event</code>, is passed to the function. This contains common event information based on jQuery or MooTools depending on which library is used as the base for Highcharts. The default action is to toggle the visibility of the series. This can be prevented by returning <code>false</code> or calling <code>event.preventDefault()</code>.
