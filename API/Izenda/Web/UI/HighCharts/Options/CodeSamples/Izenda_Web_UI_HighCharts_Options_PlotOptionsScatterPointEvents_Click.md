##About

**Summary:** <p></p>Fires when a point is clicked. The <code>this</code> keyword refers to the point object itself. One parameter, <code>event</code>, is passed to the function. This contains common event information based on jQuery or MooTools depending on which library is used as the base for Highcharts.<p></p><p>If the <code>series.allowPointSelect</code> option is true, the default action for the point's click event is to toggle the point's select state. Returning <code>false</code> cansels this action.</p>
